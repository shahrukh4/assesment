# Installation Steps

- Clone the repository

    
    jsjsjjs


- Open terminal and move to directory

    
    cd assesment


- Install dependancies

    
    composer install

- Install node packages


    npm install


- Run following artisan commands


    php artisan migrate
    php artisan db:seed


- Open terminal and run these 2 commands in separate tabs


    php artisan serve
    npm run dev
